# Public downloads

Create one lowercase kebab-case directory per topic:

```text
files/
└── example-topic/
    ├── README.md
    └── example-file.ext
```

Each topic README should state:

- what the files contain;
- when they were created or updated;
- the source and methodology, when relevant;
- any software needed to open them; and
- the license or permitted use.
