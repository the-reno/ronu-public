# Ronu public files

Files intentionally shared for public access without requiring a login.

## Structure

```text
files/       Downloadable public files, grouped by topic
README.md    Repository purpose and usage
```

## Publishing rules

- Everything committed here is public and may be downloaded, copied, or indexed.
- Review every file for personal, confidential, licensed, or proprietary information before upload.
- Do not upload credentials, private financial records, personal identifiers, internal drafts, or restricted datasets.
- Use descriptive lowercase kebab-case filenames.
- Add a short description, source, date, and usage terms alongside each shared file.

Website source belongs in [`the-reno/the-reno.github.io`](https://github.com/the-reno/the-reno.github.io). Private models and drafts belong in the private `the-reno/ronu-private` repository.
